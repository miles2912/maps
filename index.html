Hex Map Tool Help

Terrain

  • Select Tool = Terrain.
  • Choose a terrain type.
  • Click a hex to apply terrain.
  • Choose Terrain = None and click a hex to clear terrain.

Settlements

  • Select Tool = Settlement.
  • Choose Hamlet, Village, Town, City, or Capital.
  • Click a hex to place a settlement.
  • Choose Settlement = None and click a hex to remove a settlement.
  • Right-click a settlement marker to add or edit its name.

Roads and Rivers

  • Select Tool = Line.
  • Choose Road or River.
  • Click to place points.
  • Double-click to finish the line.
  • Roads and rivers are saved with the map.

Text Labels

  • Select Tool = Text.
  • Click the map to place a text label.
  • Text labels move correctly when the hex size changes.
  • Text labels are saved with the map.

Delete Tool

  • Select Tool = Delete.
  • Click a text label to delete it.
  • Click a settlement to delete it.
  • Click a road or river to delete it.
  • Terrain is still removed using Terrain = None.

Save and Load

  • Save Map exports the current map to JSON.
  • Load Map restores a previously saved map.
  • Save/load includes terrain, settlements, names, roads, rivers, text labels, and map settings.

Printing

  • Print generates a printable version of the map.
  • Controls and help do not appear on the printed page.

General Notes

  • Terrain, settlements, roads, rivers, and labels can coexist.
  • Changing hex size redraws the map and preserves map data.
